# RAG 智能文档问答助手

一个基于**智谱AI GLM-4-Flash**模型和**Streamlit**构建的RAG应用。支持上传PDF/Word文档，并进行智能问答。

## ✨ 功能特点
- 📄 支持上传 **PDF** 和 **Word (.docx)** 格式文档。
- 🧠 基于 **智谱AI** 的 Embedding 和 Chat 模型，提供准确的语义理解与生成。
- 💬 简洁美观的 Web 界面，支持相似度匹配和原文引用展示。
- ⚡️ 纯 Python 实现，依赖轻量，易于部署。

## 🛠️ 技术栈
- **前端/界面**: Streamlit
- **后端**: Python
- **AI 模型**: 智谱AI (glm-4-flash, embedding-2)
- **向量检索**: 自定义余弦相似度算法
